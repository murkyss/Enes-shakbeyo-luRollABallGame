# EnesİshakbeyoğluRollABallGame

In this Unity tutorial,

I learned how to set up the system needed to make the ball move,
how to define the area the ball travels over and apply a material to it,
how to create the walls needed to prevent the ball from falling down,
how to add collectible objects,
how to lock the camera to the player and make it move with the player,

and I didn’t encounter any errors.
